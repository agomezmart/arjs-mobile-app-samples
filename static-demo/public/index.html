These demos work in mobile Safari, but they do not work as a PWA/pinned app or inside Cordova. The goal of this project is to get similar demos working inside mobile apps using Cordova et al.

Location-based AR Demo

  1. Run npm run start
  2. Navigate to https://whatever.ngrok.io/location on your phone
  3. Move the phone around and watch it work.

Marker-based AR Demo

  1. Run npm run start
  2. Navigate to https://whatever.ngrok.io/marker on your phone
  3. Load the Hiro image in your desktop browser
  4. Point the phone at the marker and watch it work.
